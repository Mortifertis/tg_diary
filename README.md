# tg_diary

Telegram-бот дневника с ежедневными, еженедельными и ежемесячными напоминаниями.

## Возможности
- Короткие ежедневные записи (1–3 предложения) с сохранением настроения.
- Еженедельные и ежемесячные ретроспективы.
- Гибкие напоминания: ежедневные, еженедельные и ежемесячные.
- Отдельное меню **«Настройки»** в главном меню бота.
- Управление базой ежедневных вопросов пользователя:
  - просмотр полного списка;
  - добавление своих вопросов;
  - удаление вопросов;
  - постановка вопроса на паузу и возобновление.
- Все изменения настроек и вопросов сохраняются в базе данных пользователя.
- Просмотр последних записей и экспорт архива.
- Отдельный сценарий редактирования и удаления записей по индексу.
- Базовая статистика (общее число записей, серия, настроение).

## Главное меню
Сейчас в главном меню доступны:
- `Сделать новую запись`
- `Настройки`
- `Посмотреть записи`
- `Редактирование/удаление записей`

## Быстрый старт (локально)
Рекомендуется Python 3.12.x (для 3.13 потребуется установленный Rust/Cargo из-за сборки pydantic-core).
1. Создайте и активируйте виртуальное окружение.
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Создайте файл `.env` по примеру `.env.example` и заполните токен.
4. Запустите бота:
   ```bash
   python -m app.main
   ```

## Переменные окружения
Пример доступен в `.env.example`.
- `BOT_TOKEN` — токен Telegram-бота.
- `DATABASE_URL` — строка подключения (по умолчанию SQLite).
- `DEFAULT_TIMEZONE` — часовой пояс пользователя по умолчанию (например, `Europe/Moscow`).
- `DEFAULT_DAILY_TIME` — время ежедневной записи (HH:MM).
- `DEFAULT_WEEKLY_DAY` — день недели (0=пн ... 6=вс).
- `DEFAULT_WEEKLY_TIME` — время еженедельной записи (HH:MM).
- `DEFAULT_MONTHLY_DAY` — день месяца.
- `DEFAULT_MONTHLY_TIME` — время ежемесячной записи (HH:MM).
- `REMINDER_EVENING_HOUR` — час вечернего напоминания.

## Docker
1. Заполните `.env.docker` и укажите токен бота.
2. Запустите:
   ```bash
   docker compose up --build
   ```

## Команды бота
- `/start` — регистрация и открытие главного меню.
- `/time HH:MM` — установить ежедневное время напоминания.
- `/weekly <day> HH:MM` — установить еженедельное напоминание.
- `/monthly <day> HH:MM` — установить ежемесячное напоминание.
- `/pause N` — поставить напоминания на паузу на `N` дней.
- `/resume` — снять паузу напоминаний.
- `/stats` — базовая статистика.
- `/status` — текущие параметры напоминаний и паузы.
